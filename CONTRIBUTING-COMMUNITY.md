# 🏴‍☠️ Community Contributing Guide

**No coding skills required!** This guide is for fans who want to contribute lore, locations, and knowledge to the One Piece Lore Map.

## Ways to Contribute

### 1. 🐛 GitHub Issues (Recommended)

The easiest way to contribute is through GitHub Issues:

1. Go to our [Issues page](https://github.com/HoxtaH/One-Piece-Lore-Map/issues/new/choose)
2. Choose a template:
   - **🏝️ New Location** — Suggest a new island or place
   - **📜 Lore Correction** — Fix incorrect information
3. Fill out the form and submit

No account setup beyond a GitHub account is required.

---

### 2. 💬 GitHub Discussions (Community chat)

Join our [Discussions](https://github.com/HoxtaH/One-Piece-Lore-Map/discussions) to:

- Discuss lore accuracy
- Suggest features
- Help other contributors
- Share One Piece knowledge

---

## What Makes a Good Contribution?

### ✅ DO:

- **Be accurate** — Verify information against the manga/anime
- **Be specific** — Include chapter/episode references when possible
- **Be complete** — The more detail, the better!
- **Be respectful** — This is a community project
- **Be patient** — Reviews take time

### ❌ DON'T:

- Submit unconfirmed fan theories as facts
- Include spoilers without warning
- Copy text from other wikis (write in your own words)
- Submit incomplete joke entries
- Spam multiple submissions

## Location Data Guide

When contributing a location, here's what to include:

### Required Fields

| Field | Description | Example |
|-------|-------------|---------|
| **Name** | Official location name | Wano Country |
| **Region** | Part of the One Piece world | New World |
| **Description** | 2–3 sentence overview | "An isolated nation..." |

### Optional (But Helpful!) Fields

| Field | Description |
|-------|-------------|
| **History** | Major events, backstory |
| **Culture** | Traditions, society, lifestyle |
| **Notable People** | Important characters |
| **Economy** | Industries, trade goods |
| **Food** | Local cuisine, dining customs |
| **Transportation** | How people get around |
| **Quick Facts** | Fun trivia for hover popups |
| **Videos** | YouTube clips of key moments |

---

## How to Find YouTube Video IDs

We display YouTube videos for each location. Here's how to get a video ID:

1. Find a video on YouTube
2. Look at the URL: `https://www.youtube.com/watch?v=dQw4w9WgXcQ`
3. The ID is the part after `v=`: **dQw4w9WgXcQ**

**Good videos to add:**
- Epic battle scenes
- Emotional moments
- Lore explanations
- Scenic shots of the location

## Review Process

1. **Submission** — Open a GitHub Issue using a template
2. **Review** — Maintainers check for accuracy and completeness
3. **Approval/Feedback** — Added to the map or feedback requested
4. **Live!** — Your contribution appears on the map
