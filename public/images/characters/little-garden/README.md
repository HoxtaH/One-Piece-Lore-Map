## Image Guidelines

- **Format**: JPG
- **Dimensions**: 400x400px recommended
- **Aspect Ratio**: 1:1 (square)
- **File Size**: Under 200KB for web performance

## Notes

- Giants should show their distinctive colors (blue/red)
- Mr. 3's number "3" hairdo is iconic
- Island Eater should show its massive scale
- Usopp and Nami images should reflect Little Garden context
