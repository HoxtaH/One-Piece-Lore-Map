# Amazon Lily Character Images

This directory contains character images for Amazon Lily - the all-female warrior island in the Calm Belt.

## Required Images

### Kuja Royalty
- `boa-hancock.jpg` - Boa Hancock (Pirate Empress, most beautiful woman, Love-Love Fruit)
- `sandersonia.jpg` - Boa Sandersonia (Gorgon sister, Snake-Snake Fruit Anaconda)
- `marigold.jpg` - Boa Marigold (Gorgon sister, Snake-Snake Fruit King Cobra)
- `gloriosa.jpg` - Gloriosa/Elder Nyon (tiny former empress, wise elder)

### Kuja Warriors
- `marguerite.jpg` - Marguerite (kind warrior, Luffy's friend, archer)
- `sweet-pea.jpg` - Sweet Pea (Marguerite's friend, cautious warrior)
- `aphelandra.jpg` - Aphelandra (giant tall woman, gentle giant)
- `ran.jpg` - Ran (ship guard, traditional fierce warrior)

### Special Characters
- `luffy.jpg` - Monkey D. Luffy (first man on island, Hancock's love)
- `salome.jpg` - Salome (Hancock's giant snake companion/throne)
- `bacura.jpg` - Bacura (black panther execution beast, one-punched)

## Image Guidelines

- **Format**: JPG preferred
- **Dimensions**: Minimum 400x400px, square aspect ratio recommended
- **Quality**: High resolution, clear character focus
- **Content**: 
  - Hancock: Beautiful, regal, elegant pose, heart eyes for Luffy optional
  - Sandersonia: Orange hair, anaconda features or transformation
  - Marigold: Green hair, cobra features or transformation
  - Gloriosa: Tiny elderly woman with snake staff
  - Marguerite: Blonde, kind expression, bow and arrows
  - Sweet Pea: Purple hair, archer stance
  - Aphelandra: Very tall, gentle giant appearance
  - Luffy: Amazon Lily context, confused or eating mushrooms
  - Salome: Giant white snake with skull pattern
  - Bacura: Massive black panther

## Naming Convention

Use lowercase, hyphenated filenames matching the JSON data:
- Character names as they appear in `notablePeople[].avatar`
- Example: `boa-hancock.jpg`, `sweet-pea.jpg`

## Usage

These images are used in:
- Character cards on location pages
- Character modals with details
- Grid displays on `/locations/amazon-lily` page
- Referenced in `data/locations/amazon-lily.json`
