## Image Guidelines

- **Format**: JPG preferred
- **Dimensions**: Minimum 400x400px, square aspect ratio recommended
- **Quality**: High resolution, clear character focus
- **Content**: 
  - Magellan: Demon horns, poison dripping, imposing
  - Hannyabal: Egyptian headpiece, naginata, bumbling but heroic
  - Ivankov: MASSIVE head, purple hair, flamboyant, "HEEHAW!"
  - Bon Clay: Swan theme, ballet pose, heroic sacrifice moment
  - Jinbe: Whale shark features, kimono, dignified
  - Crocodile: Hook hand, scar, cigar, coat
  - Buggy: Clown makeup, red nose, scared expression
  - Shiryu: Serious swordsman, cigar, murderous

## Naming Convention

Use lowercase, hyphenated filenames matching the JSON data:
- Character names as they appear in `notablePeople[].avatar`
- Example: `bon-clay.jpg`, `daz-bones.jpg`

## Usage

These images are used in:
- Character cards on location pages
- Character modals with details
- Grid displays on `/locations/impel-down` page
- Referenced in `data/locations/impel-down.json`
