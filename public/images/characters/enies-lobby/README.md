## Image Guidelines

- **Format**: JPG preferred
- **Dimensions**: Minimum 400x400px, square aspect ratio recommended
- **Quality**: High resolution, clear character focus
- **Content**: 
  - Show character during Enies Lobby arc when possible
  - For Luffy: Gear 2nd steam effect or Gear 3rd giant form
  - For Zoro: Asura demonic aura
  - For Sanji: Fire kick effects
  - For Chopper: Monster Point preferred
  - For Sogeking: Full mask/cape costume
  - For Robin: Emotional "I want to live!" scene
  - For CP9: Show Zoan forms or battle stances
  - For Merry: Spirit/Klabautermann appearance

## Naming Convention

Use lowercase, hyphenated filenames matching the JSON data:
- Character names as they appear in `notablePeople[].avatar`
- Example: `rob-lucci.jpg`, `going-merry.jpg`

## Usage

These images are used in:
- Character cards on location pages
- Character modals with details
- Grid displays on `/locations/enies-lobby` page
- Referenced in `data/locations/enies-lobby.json`
