# Reverse Mountain - Character Images

This directory contains character avatar images for Reverse Mountain location.

## Required Images

### Main Characters
- `laboon.jpg` - Island whale waiting for the Rumbar Pirates
- `crocus.jpg` - Lighthouse keeper and former Roger Pirates doctor
- `yorki.jpg` - Former Rumbar Pirates captain (deceased)
- `brook.jpg` - Rumbar Pirates musician (connection to Laboon)

## Image Specifications
- Format: JPG or PNG
- Recommended size: 400x400px minimum
- Aspect ratio: 1:1 (square) preferred
- Quality: High resolution for character detail

## Naming Convention
Use lowercase with hyphens for multi-word names:
- ✅ `laboon.jpg`
- ✅ `crocus.jpg`
- ❌ `Laboon.jpg`
- ❌ `crocus_image.jpg`
