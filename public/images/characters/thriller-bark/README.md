# Thriller Bark Character Images

This directory contains character images for Thriller Bark - the world's largest pirate ship and Moria's zombie factory.

## Required Images

- `brook.jpg` - Brook (skeleton musician who joins the crew)
- `nightmare-luffy.jpg` - Nightmare Luffy (100 shadows absorbed, blue giant)
- `zoro.jpg` - Roronoa Zoro ("Nothing Happened" moment, Shusui)
- `sanji.jpg` - Sanji (fought Absalom)
- `nami.jpg` - Nami (almost bride, weather attacks)
- `usopp.jpg` - Usopp (defeated Perona with negativity)

### Mysterious Four (Main Villains)

### Zombies & Special Characters

## Image Guidelines

- **Format**: JPG preferred
- **Dimensions**: Minimum 400x400px, square aspect ratio recommended
- **Quality**: High resolution, clear character focus
- **Content**: 
  - Brook: Skeleton with afro, gentleman pose, musical instrument
  - Nightmare Luffy: Blue-skinned giant form, overwhelming presence
  - Zoro: Standing in blood with "Nothing Happened" vibe or with Shusui
  - Moria: Large with horns, scissors, shadow powers
  - Perona: Gothic Lolita style, ghosts around her
  - Absalom: Partially invisible or grotesque modifications
  - Hogback: Mad scientist appearance
  - Ryuma: Samurai zombie with Shusui
  - Oars: Massive giant zombie
  - Kuma: Bible, paw gloves, stoic expression

## Naming Convention

Use lowercase, hyphenated filenames matching the JSON data:
- Character names as they appear in `notablePeople[].avatar`
- Example: `nightmare-luffy.jpg`, `dr-hogback.jpg`

## Usage

These images are used in:
- Character cards on location pages
- Character modals with details
- Grid displays on `/locations/thriller-bark` page
- Referenced in `data/locations/thriller-bark.json`
