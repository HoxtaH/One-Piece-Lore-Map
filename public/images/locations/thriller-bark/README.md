# Thriller Bark Location Images

This directory contains sub-location images for Thriller Bark - the massive island-ship.

## Required Images

- `mansion-entrance.jpg` - Mansion Entrance (gothic gates, zombie guards, Cerberus)
- `morias-chamber.jpg` - Moria's Chamber (throne room, shadow storage)
- `laboratory.jpg` - Hogback's Laboratory (operating tables, zombie creation)
- `wonder-garden.jpg` - Perona's Wonder Garden (cute gothic room, negative hollows)
- `graveyard.jpg` - Graveyard (zombie spawning, fog, tombstones)
- `freezer.jpg` - Freezer (corpse storage, Oars chamber, sub-zero)
- `oars-zone.jpg` - Oars Battle Zone (giant battle, crew teamwork)
- `nothing-happened.jpg` - Nothing Happened Spot (Zoro's sacrifice, blood pool)

## Image Guidelines

- **Format**: JPG preferred
- **Dimensions**: Minimum 800x600px, landscape preferred
- **Quality**: High resolution establishing shots
- **Content**: Show the horror-gothic atmosphere
  - Mansion: Dark, foreboding, haunted house vibes
  - Moria's Chamber: Massive throne, shadows everywhere
  - Laboratory: Mad science, body parts, operating theater
  - Wonder Garden: Cute but creepy, stuffed animals, pink/gothic
  - Graveyard: Fog, tombstones, zombies emerging
  - Freezer: Ice, frost, massive frozen bodies
  - Oars Zone: Destruction, giant scale
  - Nothing Happened: Blood pool, isolated area, sacred moment

## Naming Convention

Use lowercase, hyphenated filenames matching the JSON data:
- Location names as they appear in `subLocations[].image`
- Example: `nothing-happened.jpg`, `wonder-garden.jpg`

## Usage

These images are used in:
- Sub-location showcase sections
- Location detail views
- Exploration interface
- Referenced in `data/locations/thriller-bark.json`
