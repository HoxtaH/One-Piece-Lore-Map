# Long Ring Long Land Location Images

This directory contains location images for the Long Ring Long Land sub-locations.

## Required Images

- `ring-islands.jpg` - Ten ring-shaped islands connected at low tide
- `tonjits-camp.jpg` - Abandoned camp with extremely tall stilts
- `davy-back-arena.jpg` - Competition grounds for Davy Back Fight
- `bamboo-forest.jpg` - Dense stretched bamboo forests
- `frozen-ocean.jpg` - Ocean frozen solid by Aokiji's Ice Age

## Image Guidelines

- **Format**: JPG
- **Dimensions**: 800x600px or 1200x800px recommended
- **Aspect Ratio**: 4:3 or 3:2 preferred
- **File Size**: Under 500KB for web performance

## Notes

- Emphasize vertical stretching in all geography
- Ring-shaped island formation is unique
- Animals and plants should appear elongated
- Frozen ocean should show Aokiji's overwhelming power
- Bamboo forests with extremely tall stalks
- Davy Back Fight arena with game setups
- Coastal areas showing tidal connections
