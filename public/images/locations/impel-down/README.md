# Impel Down Location Images

This directory contains sub-location images for Impel Down - the six levels of hell.

## Required Images

- `level-1.jpg` - Level 1 Crimson Hell (blade grass forest, cutting torture)
- `level-2.jpg` - Level 2 Wild Beast Hell (monsters, Basilisks, Sphinx, Jailer Beasts)
- `level-3.jpg` - Level 3 Starvation Hell (desert, heat, dehydration)
- `level-4.jpg` - Level 4 Blazing Hell (boiling blood pool, intense flames)
- `level-5.jpg` - Level 5 Freezing Hell (frozen wasteland, wolves, extreme cold)
- `newkama-land.jpg` - Level 5.5 Newkama Land (colorful paradise, Okama kingdom, hidden)
- `level-6.jpg` - Level 6 Eternal Hell (darkness, worst criminals, legendary prisoners)
- `gates-of-justice.jpg` - Gates of Justice (massive gates, Bon Clay's sacrifice location)

## Image Guidelines

- **Format**: JPG preferred
- **Dimensions**: Minimum 800x600px, landscape preferred
- **Quality**: High resolution establishing shots
- **Content**: Show the hellish/torture atmosphere progression
  - Level 1: Red bladed grass, cutting atmosphere
  - Level 2: Monsters, beasts, animal terror
  - Level 3: Desert heat, starvation, suffering
  - Level 4: Fire, boiling blood, extreme heat, red/orange
  - Level 5: Ice, snow, frozen, blue/white, wolves
  - Newkama Land: CONTRAST - colorful, fun, paradise, rainbow
  - Level 6: Dark, ominous, cells, worst criminals
  - Gates: Massive mechanical gates, control room area

## Naming Convention

Use lowercase, hyphenated filenames matching the JSON data:
- Location names as they appear in `subLocations[].image`
- Example: `newkama-land.jpg`, `gates-of-justice.jpg`

## Usage

These images are used in:
- Sub-location showcase sections
- Location detail views
- Exploration interface
- Referenced in `data/locations/impel-down.json`
