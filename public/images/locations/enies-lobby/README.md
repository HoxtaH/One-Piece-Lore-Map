# Enies Lobby Location Images

This directory contains sub-location images for Enies Lobby - the Judicial Island.

## Required Images

- `main-gate.jpg` - Main Island Gate (Rocket Man crash site)
- `courthouse.jpg` - Courthouse (mock trial chambers, Marine battalions)
- `tower-of-justice.jpg` - Tower of Justice (Robin held here, CP9 battles, flag burning)
- `bridge-of-hesitation.jpg` - Bridge of Hesitation (massive bridge, waterfall views)
- `underground-passage.jpg` - Underground Passage (secret tunnels, chase sequences)
- `gates-of-justice.jpg` - Gates of Justice (massive gates, Marine HQ connection)
- `buster-call-zone.jpg` - Buster Call Destruction Zone (ten battleships firing)

## Image Guidelines

- **Format**: JPG preferred
- **Dimensions**: Minimum 800x600px, landscape preferred
- **Quality**: High resolution establishing shots
- **Content**: Show the scale and atmosphere of each location
  - Main Gate: Broken gates, Rocket Man crash
  - Courthouse: Grand architecture, Marine presence
  - Tower: Tall imposing structure, World Government flag
  - Bridge: Long suspension bridge over waterfall
  - Underground: Dark tunnels, prison transport vibe
  - Gates of Justice: Massive scale, ominous opening
  - Buster Call: Destruction, bombardment, battleships

## Naming Convention

Use lowercase, hyphenated filenames matching the JSON data:
- Location names as they appear in `subLocations[].image`
- Example: `tower-of-justice.jpg`, `gates-of-justice.jpg`

## Usage

These images are used in:
- Sub-location showcase sections
- Location detail views
- Exploration interface
- Referenced in `data/locations/enies-lobby.json`
