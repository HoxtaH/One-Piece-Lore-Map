# Whiskey Peak - Location Images

This directory contains sub-location images for Whiskey Peak.

## Required Images

### Sub-Locations
- `town-square.jpg` - Central party venue with feast tables
- `cactus-rocks.jpg` - Distinctive cactus-shaped rock formations
- `harbor.jpg` - Small port where ships dock
- `cemetery.jpg` - Graveyard where Zoro fought bounty hunters
- `igarams-residence.jpg` - House where Vivi's identity was revealed
- `nuns-residence.jpg` - Building where Sanji was distracted
- `rooftops.jpg` - Battle terrain for Zoro's fights

## Image Specifications
- Format: JPG or PNG
- Recommended size: 800x600px minimum
- Aspect ratio: 16:9 or 4:3 preferred
- Quality: High resolution landscape images

## Naming Convention
Use lowercase with hyphens:
- ✅ `town-square.jpg`
- ✅ `igarams-residence.jpg`
- ❌ `Town_Square.jpg`
