This directory contains location images for the Water 7 sub-locations.

- `galley-la.jpg` - Massive shipbuilding headquarters (company tower)
- `dock-1.jpg` - Main shipyard (Merry's death sentence location)
- `canals.jpg` - Multi-level waterway system (city streets)
- `franky-house.jpg` - Franky Family base (backstreets, dismantling)
- `bluenos-bar.jpg` - Tavern with secret CP9 meeting rooms
- `shift-station.jpg` - Sea Train station (Kokoro's workplace)
- `iceburgs-mansion.jpg` - Mayor's elegant home (assassination attempt site)
- `scrap-island.jpg` - Ship graveyard (Luffy vs Usopp duel)

## Image Guidelines

- **Format**: JPG
- **Dimensions**: 800x600px or 1200x800px recommended
- **Aspect Ratio**: 4:3 or 3:2 preferred
- **File Size**: Under 500KB for web performance
