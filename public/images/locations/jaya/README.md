# Jaya Location Images

This directory contains location images for the Jaya sub-locations.

## Required Images

- `mock-town.jpg` - Lawless pirate haven with taverns and chaos
- `crickets-house.jpg` - Coastal salvage headquarters with diving equipment
- `forest.jpg` - Dense jungle where South Birds live
- `tavern.jpg` - Interior of taverns (Blackbeard's speech location)
- `coast.jpg` - Coastline with salvage operations and ship modifications
- `knock-up-stream.jpg` - Massive geyser eruption launching into sky

## Image Guidelines

- **Format**: JPG
- **Dimensions**: 800x600px or 1200x800px recommended
- **Aspect Ratio**: 4:3 or 3:2 preferred
- **File Size**: Under 500KB for web performance

## Notes

- Mock Town should feel rough and lawless
- Tavern interiors should show rowdy pirate atmosphere
- Cricket's house on coastline with salvage equipment visible
- Knock Up Stream should show massive scale and power
- Island shaped like skull's profile (half missing)
- Spring island climate - lush vegetation
- Coastal areas with diving operations
