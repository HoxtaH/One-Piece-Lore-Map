# Little Garden Location Images

This directory contains location images for the Little Garden sub-locations.

## Required Images

- `dorrys-camp.jpg` - Dorry's Cave and Camp (Blue Ogre's dwelling)
- `brogys-camp.jpg` - Brogy's Cave and Camp (Red Ogre's dwelling)
- `dueling-ground.jpg` - Central battlefield (scarred by century of combat)
- `wax-house.jpg` - Mr. 3's temporary wax structure
- `jungle.jpg` - Prehistoric jungle (dinosaurs and ancient creatures)
- `volcano.jpg` - Active volcano (duel signal)
- `coastal-area.jpg` - Beach and shore (Going Merry location)
- `candle-service-set.jpg` - Giant Candle Service Set trap (wax execution platform)

## Image Guidelines

- **Format**: JPG
- **Dimensions**: 800x600px or 1200x800px recommended
- **Aspect Ratio**: 4:3 or 3:2 preferred
- **File Size**: Under 500KB for web performance

## Notes

- Show the massive scale of giant-sized locations
- Prehistoric atmosphere with dinosaurs and ancient flora
- Wax structures should look candle-like
- Volcano eruptions provide dramatic atmosphere
- Jungle density makes it dangerous for normal humans
