# Amazon Lily Location Images

This directory contains sub-location images for Amazon Lily - the island of women.

## Required Images

- `palace.jpg` - Kuja Palace (mountain top, throne room, Hancock's chambers)
- `arena.jpg` - Kuja Arena (battle grounds, Bacura's cage, execution site)
- `village.jpg` - Kuja Village (traditional houses, communal areas, daily life)
- `gorgon-shrine.jpg` - Gorgon Shrine (forbidden sacred site, cover story location)
- `jungle.jpg` - Jungle Paths (lush tropical, wild animals, training grounds)
- `harbor.jpg` - Kuja Pirates' Harbor (snake-pulled ship, Calm Belt access)

## Image Guidelines

- **Format**: JPG preferred
- **Dimensions**: Minimum 800x600px, landscape preferred
- **Quality**: High resolution establishing shots
- **Content**: Show the tropical warrior paradise atmosphere
  - Palace: Magnificent, snake motifs, mountain peak, regal
  - Arena: Large coliseum, spectator stands, battle area
  - Village: Traditional island architecture, communal, peaceful
  - Shrine: Mysterious, forbidden, sacred snake decorations
  - Jungle: Lush tropical vegetation, wild, training areas
  - Harbor: Unique snake-ship, dock area, Calm Belt waters

## Naming Convention

Use lowercase, hyphenated filenames matching the JSON data:
- Location names as they appear in `subLocations[].image`
- Example: `gorgon-shrine.jpg`, `palace.jpg`

## Usage

These images are used in:
- Sub-location showcase sections
- Location detail views
- Exploration interface
- Referenced in `data/locations/amazon-lily.json`
